GOV.UK One Login?

You will need to create a GOV.UK One Login account to access the new service.

GOV.UK One Login gives you access to all services that use GOV.UK and stores them all in one place.

Using GOV.UK One Login

It is important that you use the same email address to create your GOV.UK One Login account as you have previously used to access ICMS.

Using the same email address will mean all your data will be moved over automatically.

You will be redirected to the GOV.UK One Login page one you have selected “Start Now” from the homepage.

You will then need to:

  • use the same email address you previously used to sign in with
  • create a GOV.UK One Login account
  • choose a password

In future, you should then always use your GOV.UK One Login to sign in to this service.

GOV.UK One Login help

If you need any additional help with GOV.UK One Login, contact the GOV.UK One Login team.
